void procura_destino ( int y,
int x,
JOGO * game,
int * flag )

Auxiliar da hint. Percorre para cada carta a procura, nas outra pilhas se existe uma carta no topo, um valor acima desta. No caso de ser do mesmo naipe coloca a flag, na carta para 1 que indica que a carta será mostrada a verde. E se a carta de destino encontrada for de um naipe diferente, coloca a flag da carta de origem a 2, que indica que ficará amarela.
